List out the different ways to select an item.

CSS Selectors
Element Selector (div, p, h1) Selects all elements of a type.
Class Selector (.classname) Selects elements with a specific class.
ID Selector (#idname) Selects an element with a specific ID.

What is the difference between an ID and a Class?

ID only use once, Class can use it more than once.

What is the difference between a SPAN tag and a DIV tag?

The span tag is an inline element used for styling or modifying small parts of text within a block, while the div tag is a block-level element used for structuring and grouping larger sections of content.

What is the difference between a DIV tag and a HEADER, MAIN, or FOOTER?

A div is just a box used to group content with no special meaning.

Using header, main, and footer helps search engines and screen readers understand the page better.
