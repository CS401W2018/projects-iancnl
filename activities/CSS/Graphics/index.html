Graphics

What kind of photos are good to encode using JPEG?

Best for detailed, colorful photos like landscapes, portraits, and complex images due to high compression with minimal quality loss.

What kind of photos are good to encode using GIF?

Ideal for simple images with few colors, animations, and transparency support (limited to 256 colors).

What is unique about the PNG format?

Supports lossless compression, transparency, and is great for graphics, logos, and images needing sharp edges.

How do you resize your photo to the size it should be displayed?

Use photo editing software (e.g., Photoshop, GIMP) to adjust dimensions to the exact display size.

Why should you resize photos using photo editing software instead of resizing it using CSS?

CSS resizing only scales the image visually, keeping the original file size, which can slow loading and reduce quality.
