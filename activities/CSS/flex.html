What you learned about Flex and CSS.

CSS is a stylesheet language used to style HTML elements, controlling properties like color, spacing, and layout. Flexbox is a CSS layout module designed specifically for arranging items efficiently in a row or column, making it easier to align and distribute space dynamically. While CSS includes many layout techniques, Flexbox is specialized for flexible, responsive designs.

Why do you need to have an accessible website? What does that mean?

An accessible website ensures that all users, including those with disabilities, can navigate, understand, and interact with its content effectively. This improves inclusivity, user experience, and compliance with legal standards.
