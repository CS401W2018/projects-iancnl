Explain the difference between inline, embedded, and external CSS

Inline CSS: Applied directly to an HTML element using the style attribute.

Embedded CSS: Placed within a style tag inside the HTML head. Affects multiple elements in the same document.

External CSS: Stored in a separate .css file and linked using link. Best for maintaining consistency across multiple pages.

Explain why you should only use external CSS.

External CSS is the best choice because it keeps HTML clean, improves maintainability, and ensures consistency across multiple pages. It also enhances performance by allowing browsers to cache styles, reducing page load time. Additionally, it makes collaboration easier and separates content from design, following best coding practices.

What is a selector used for?

A selector is used in CSS to target and style specific HTML elements. It defines which elements the styles will apply to based on their tag name, class, ID, attributes, or relationships to other elements.
