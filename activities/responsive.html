Why is the viewport meta tag important?

The viewport meta tag makes websites responsive by controlling how they display on different screen sizes, especially mobile devices.

How can you adjust the CSS for printing?

Use the media print rule in CSS to define styles specifically for printed content.

When would you want to adjust the CSS for printing?

When you need to optimize layout, hide unnecessary elements, or improve readability for printed documents.

Why do we need a viewport meta tag?

Viewport meta tag makes web pages responsive on different screen sizes, especially mobile.

When you select the same tag twice, and in one of them you say "flex-direction: column;" and the other says "flex-direction: row;", which one takes priority?

Conflicting CSS rules is the last rule in the stylesheet takes priority.

How do you change the CSS for a web page displayed on a phone?

Use media queries like @media (max-width: 768px) {} to apply phone-specific styles.
